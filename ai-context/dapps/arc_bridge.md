## Description

The official bridge for moving assets between Ethereum mainnet and Arc Network. Supports USDC, EURC, and other whitelisted stablecoins.

## Allowed Actions

- `bridge_in` - Bridge assets from Ethereum to Arc
- `bridge_out` - Bridge assets from Arc to Ethereum
- `claim_bridged` - Claim bridged assets on destination chain

## Key Features

1. **Native USDC Support** - Direct USDC bridging with Circle integration
2. **Fast Finality** - Leverage Arc's deterministic finality
3. **Multi-Token** - Support for major stablecoins
4. **Secure** - Industry-standard bridge security
