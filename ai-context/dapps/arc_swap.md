# Arc Swap

## Basic Information

| Property | Value |
|----------|-------|
| Name | Arc Swap |
| Category | Trading |
| Official Link | https://swap.arc.network |

## Description

Native DEX for Arc Network with concentrated liquidity and efficient swaps. Optimized for stablecoin pairs with minimal slippage.

## Allowed Actions

- `swap` - Swap tokens directly
- `add_liquidity` - Provide liquidity to pools
- `remove_liquidity` - Remove liquidity
- `collect_fees` - Collect earned trading fees

## Key Features

1. **Concentrated Liquidity** - Capital-efficient LP positions
2. **Stablecoin Optimized** - Minimal slippage on stable pairs
3. **USDC Gas** - Predictable swap costs
4. **Fast Execution** - Sub-second trade confirmations

## Campaign Use Cases

- Trading tutorial campaigns
- Liquidity provision incentives
- Swap volume competitions
