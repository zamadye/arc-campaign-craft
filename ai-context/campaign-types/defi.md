## Tone Guidelines

| Aspect | Do | Don't |
|--------|-----|-------|
| Benefits | Focus on UX improvements | Promise financial returns |
| Technology | Explain what's different | Make unverified claims |
| Ecosystem | Mention real partners | Exaggerate partnerships |
| Urgency | Explain genuine value | Create false FOMO |

## Required Elements

1. **Must mention** @ArcFlowFinance or Arc Network
2. **Should include** specific DeFi use case
3. **Can include** comparison to other chains (factually)
4. **Must avoid** financial promises, airdrop mentions

## Caption Structure

```
[Hook - specific benefit] + [How Arc makes it possible] + [Call to action] + #ArcNetwork + [optional 2nd hashtag]
```
