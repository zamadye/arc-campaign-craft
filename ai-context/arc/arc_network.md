# Arc Network - Official Knowledge Base

## Overview

**Arc Network** is the **Economic OS for the internet** - a purpose-built EVM-compatible Layer-1 blockchain designed for stablecoin finance and tokenization.

## Core Value Proposition

Arc is advancing the frontier of stablecoin finance with:
- **USDC as native gas** - Eliminates volatility, enables predictable costs
- **Deterministic settlement finality** - Sub-second, final confirmations
- **Opt-in privacy** - Privacy-preserving transactions when needed
- **Stable transaction fee architecture** - No gas spikes

## Technical Foundation

### Malachite Consensus
Arc's bespoke consensus layer provides:
- Deterministic finality (no probabilistic waiting)
- High throughput for value transfer
- Byzantine fault tolerance
- Energy efficiency

### EVM Compatibility
- Deploy existing Solidity contracts without modification
- Full toolchain support: Hardhat, Foundry, Remix
- All standard EVM opcodes supported
- Familiar development experience
