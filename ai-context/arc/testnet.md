# Arc Testnet - Campaign Context

## Current Status

The Arc Public Testnet is **LIVE** and actively accepting:
- Developer deployments
- User testing
- Ecosystem integration testing

## Key Testnet Features

### USDC as Gas
- Testnet uses USDC for all transaction fees
- No volatile native token required
- Faucet available for testnet USDC

### Fast Finality
- Transactions confirm in sub-second timeframes
- No waiting for block confirmations
- Deterministic - when confirmed, it's final

### Full EVM Tooling
- Hardhat, Foundry, Remix all work
- Standard Ethereum tooling compatible
- Familiar development experience

## Campaign Themes for Testnet

### Encouraged Narratives
1. "Experience sub-second finality firsthand"
2. "Pay gas in USDC - no volatile tokens"
3. "Deploy your contracts on the next-gen L1"
4. "Help stress test Arc before mainnet"

### Discouraged Narratives
- Promising mainnet rewards for testnet activity
- Implying guaranteed airdrops
- Comparing token prices (Arc has no speculative token)
- Making unverified claims about performance
